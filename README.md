# katabankocr

Clojure implementation of the [Kata Bank OCR] [1]
[1]: http://codingdojo.org/cgi-bin/wiki.pl?KataBankOCR


## Usage

To execute with test.data:

```lein run test.data```

To save the results in a file:

```lein run test.data > test.out```

To run tests:
```lein midje```
